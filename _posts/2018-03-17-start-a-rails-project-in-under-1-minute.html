First of all, make sure you have ruby, sqlite3 and rails installed on your machine.

ruby -v
sqlite3 --version

We need sqlite for the developement environment. You can use any other database if you want but Rails uses sqlite by default so we are going to keep it for simplicity.

rails -v

Go to your projects folder and run:

rails new blog

This command will create a fresh project folder filled with all the necessary files. Go to that folder and run:

rails s

to start a server.

Open up your browser and go to "localhost:3000". You will see that our Rails app works as intended.
